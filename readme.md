# 📋 Formulário Multi-Etapas com Validação e Integração (Formspree)

Projeto de formulário moderno desenvolvido com **HTML, CSS e JavaScript puro**, focado em **experiência do usuário (UX)** e **captação de leads**.

---

## 🚀 Demonstração

> Formulário dividido em etapas com validação em tempo real, máscara de telefone e envio para e-mail via Formspree.

---

## ✨ Funcionalidades

* ✅ Formulário multi-etapas (step by step)
* ✅ Barra de progresso dinâmica
* ✅ Validação de campos obrigatórios (sem uso de alert)
* ✅ Feedback visual de erros
* ✅ Máscara automática para telefone
* ✅ Validação de telefone com 11 dígitos (DDD + número)
* ✅ Seleção de motivo de contato
* ✅ Modal de sucesso com animação
* ✅ Integração com envio de e-mail via Formspree
* ✅ Layout responsivo

---

## 🧠 Tecnologias utilizadas

* HTML5
* CSS3
* JavaScript (Vanilla JS)
